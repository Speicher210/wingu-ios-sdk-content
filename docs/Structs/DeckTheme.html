DeckTheme

public struct DeckTheme

Allows to set the global theme (colors, fonts, etc.) that will be displayed in DeckView.
As a default, the configuration from the DeckTheme will be applied in all ComponentComposition subclasses. Overriding these variables in this configuration will apply to all compositions.
If there is a need to change certain details of the components, that can be done in the ComponentComposition subclasses.

  • actionColor
    • actionColor: The color used in the background of elements containing actions. The same color will be used as the text tint color in the text element.

    Default value: #0C1E3D

    Declaration

    Swift

    public var actionColor: UIColor
  • actionColorText
    • actionColorText: The text color of the elements displayed at actionColor.
      The text tint component color is excluded and uses actionColor for tint instead.

    Default value: #FFFFFF

    Declaration

    Swift

    public var actionColorText: UIColor
  • isRoundedCornersButton
    • isRoundedCornersButton: The Boolean value that shows the elements in DeckView must have sharp or rounded corners.

    Default is true

  • fontName
    • fontName: The font object that is used in all components in DeckView.

    The size is a reference size that can be upscaled when needed.

    Note: If you require a custom font you need to add it in your application bundle and link it there.

    Default value: System font of size 12
