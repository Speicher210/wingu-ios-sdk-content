  • BrandBarComposition

    Styling of BrandBarComponent.

    Most of the styling including background and font color is done in wingu portal. Font settings will apply only for font family and size.

    Note: Color will be overriden by with color settings in wingu portal.

    Parameters configurable in wingu portal:

    • Font color
    • Background color
    • Mode (Image + Text, Image only, Text only)
