  • WinguComposition

    DeckView is described by multiple parameters depends on component. Use WinguComposition to use custom component display components in custom styling. Every component has custom set of parameters that needs to be set if customization is needed.

    If only basic configuration is needed DeckTheme provides basic customization options.

    Values can be override with custom composition or replaced single parameter from .defaultComposition static variable.

  • FontComposition

    Contains information about font used in DeckView.
    Should be always be used with one of the initializers. Default values if not set:

    • fontName: systemFontName (Depends on iOS version could be SanFrancisco or Helvetica)
    • fontColor: #0c1e3d
    • fontSize: 12
