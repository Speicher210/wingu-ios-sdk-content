DeckViewDelegate

public protocol DeckViewDelegate : AnyObject

Callback for action performed on DeckView.
Uses UserInfoKey to pass informations about action.

  • actionPerformed(_:payload:userInfo:)

    Whenever user action is done on DeckView this callback swill be called.

    See UserInfoKey for detailed information about actions that can be done in Deck

    It can be used with DeckView.performDefault(:) method if there is no need to override any of the actions.
