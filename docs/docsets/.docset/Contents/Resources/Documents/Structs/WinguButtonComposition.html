WinguButtonComposition

public struct WinguButtonComposition

This is the default button used everywhere in the DeckView.
It is possible that it only contains text.

  • backgroundColor

    This is the background color of the button.

    Default value is #0061D6

    Declaration

    Swift

    public var backgroundColor: UIColor
  • font

    The FontComposition is the value for the text shown on the button.

    The default value is font size 12 in white

    Declaration

    Swift

    public var font: FontComposition
  • bordered

    Denotes the borders as visible on the button.
    Default value is false.

    Declaration

    Swift

    public var bordered: Bool
  • roundedCorners

    Denotes that the corners should be rounded.
    When this flag is set to true the corners are always rounded, dividing the height by 2. (ex: The height of the button is 100, therefore the corner radius will be 50).

    Default value is true.

    Declaration

    Swift

    public var roundedCorners: Bool
