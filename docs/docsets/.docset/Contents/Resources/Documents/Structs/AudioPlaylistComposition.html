    If available, this denotes the song title that is displayed.

    Default value: System font of size 12, color: #FFFFFF

    This denotes the album name, which is displayed when available.

    Default value: System font of size 10, color: #FFFFFF

    This denotes the playlist track list view.
    It will displayed on a track list as the title of a cell. The value will be the title of a song from the IDv3 tag of the mp3 file. Default value: System font of size 12, color: #0C1E3D

    This denotes the playlist subtitle in the track list view.
