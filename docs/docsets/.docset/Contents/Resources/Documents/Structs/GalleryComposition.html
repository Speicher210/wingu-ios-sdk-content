GalleryComposition

public struct GalleryComposition

Denotes the styling of the GalleryComponent.

The styles will be applied solely to the in-line version of the gallery. For information concerning the fullscreen gallery, please visit our open-source project WinguGallery

  • shouldShowPageIndication

    When a gallery has more than one image, a gallery page indicator will be displayed. Setting the default to false will not show the indicator.
    By default is set to true

    Declaration

    Swift

    public var shouldShowPageIndication: Bool
  • pageControlTextLabel

    Denotes the text label below the images. When the variable shouldShowPageIndication is set to false it will not be shown.
    The default is set to System font of size 14, color: white

    Declaration

    Swift

    public var pageControlTextLabel: FontComposition
  • pageControlGradientColor

    Denotes the fade-out for the page indication. It witll take no effect when variable shouldShowPageIndication is set to false.
    The default value is set to #FFFFFF

    Declaration

    Swift

    public var pageControlGradientColor: UIColor
  • descriptionOfImage

    Denotes the text label below the image. The description of the image can be provided in the wingu portal.
    Default value: System font of size 14, color: #0C1E3D, alpha: 0.5
