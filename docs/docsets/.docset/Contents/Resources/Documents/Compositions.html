  • WinguComposition

    Use WinguComposition to customize your component display style. Every component has a custom set of parameters. These parameters can be customized when required.

    If only basic configuration is required DeckTheme provides several basic customization options.

    Overriding values is possible with the custom composition, or can be replaced with a single parameter from the .defaultComposition static variable.

  • FontComposition

    Contains information about font used in DeckView.
    This should always be used with one of the initializers.
    The default values are as follows:

    • fontName: systemFontName (Depending on the iOS version this is either SanFrancisco or Helvetica)
    • fontColor: #0c1e3d
    • fontSize: 12
