FontComposition

public struct FontComposition : Hashable

Contains information about font used in DeckView.
Should be always be used with one of the initializers. Default values if not set:

  • fontName: systemFontName (Depends on iOS version could be SanFrancisco or Helvetica)
  • fontColor: #0c1e3d
  • fontSize: 12
  • font

    UIFont object combined with parameters from initializers.

    Note: If font will not be found default system font of size 12 will be returned
