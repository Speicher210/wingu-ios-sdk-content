FormComposition

public struct FormComposition

Denotes the styling of the FormComponent.

Example view of component:

  • checkboxTintColor

    Denotes the selected tint color for the checkboxes if selected. Unselected checkboxes will remain set to the system default.
    Default value: #0061D6

    public var checkboxTintColor: UIColor
  • inputFont

    Denotes the text field font. These settings also be used for the checkbox label.
    Default value: System font of size 14, color: #0C1E3D

    Declaration

    Swift

    public var inputFont: FontComposition

    Denotes the header text for each input field, including checkboxes and dropdown menus.
    Default value: System font of size 18, color: #0C1E3D

    Declaration

    Swift

    public var indicatorFont: FontComposition
  • indicatorFontBolded

    Denotes the header text for each input field, including checkboxes and dropdown menus. This style is the default setting for required questions.
    Default value: System font of size 18, color: #0C1E3D

    Declaration

    Swift

    public var indicatorFontBolded: FontComposition
  • actionButton

    Denotes the style of the submit button.
    Default is default WinguButtonComposition style.

    Declaration

    Swift

  • actionButtonDisabled

    Denotes the style of the disabled button.
    This button will be shown whenever the form was submitted and the portal settings for the form do not allow re-submitting the form, or only allow re-submitting the form after a certain period of time.
    Default is UIColor.lightGray with UIColor.white font color.

    Declaration

    Swift

    public var actionButtonDisabled: WinguButtonComposition
