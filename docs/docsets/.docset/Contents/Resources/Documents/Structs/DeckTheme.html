DeckTheme

public struct DeckTheme

Setting global theme (colors, fonts, etc.) for displaying DeckView.
ComponentComposition classes will get information in first order from this theme struct first.

  • actionColor
    • actionColor: Color used for background on elements with actions.
      This will be also text tint color in text components.

    Default is blue

    Declaration

    Swift

    public var actionColor: UIColor
  • actionColorText
    • actionColorText: Text color of elements displayed at actionColor.
      Text component tint color is excluded and uses actionColor for tint instead.

    Default is white

    Declaration

    Swift

    public var actionColorText: UIColor
  • isRoundedCornersButton
    • isRoundedCornersButton: Boolean value that indicates is elements on DeckView should have sharp or rounded corners.

    Default is true

    Declaration

    Swift

    public var isRoundedCornersButton: Bool
  • fontName
    • fontName: Font object used in all components in DeckView.

    Size is a reference size that will be upscaled when needed.

    Note: If you need a custom font you will need provide it in your application bundle and link it here.

    Default is system font of size 10.

    Declaration

    Swift

    public var fontName: UIFont
