FontComposition

public struct FontComposition : Hashable

Contains information about font used in DeckView.
This should always be used with one of the initializers.
The default values are as follows:

  • fontName: systemFontName (Depending on the iOS version this is either SanFrancisco or Helvetica)
  • fontColor: #0c1e3d
  • fontSize: 12
  • font

    The UIFont object is combined with the parameters from the initializers.

    Note: If the font is not found, parameters will be reset to the the default font and default font size 12
